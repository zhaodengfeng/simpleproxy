# SimpleProxy

A simple multi-protocol proxy installer script.

## Features

- One-script installation
- Multiple protocol support
- Basic status and management helpers

## Requirements

- Linux server
- Root privileges

## Usage

```bash
chmod +x simpleproxy.sh
sudo ./simpleproxy.sh
```

Follow the interactive menu in the terminal.
